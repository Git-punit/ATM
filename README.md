# ATM Simulator (Java Console Application)

This is a simple Java console-based ATM simulator that allows users to:

- Log in with an account number and PIN
- View their balance
- Deposit funds
- Withdraw funds

The application uses basic Java features like classes, conditionals, loops, and a `HashMap` to simulate account storage.
